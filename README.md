# Justice Companion

A legal case management and document automation system designed to empower self-represented individuals in their fight for justice.

## Overview

Justice Companion provides tools and guidance for people representing themselves in legal matters including:
- Employment disputes (wrongful dismissal, discrimination, unpaid wages)
- Housing issues (illegal eviction, disrepair, deposit disputes) 
- Consumer claims (faulty goods, refunds, holiday disputes)
- Council challenges (benefits, planning, fines)
- Insurance and financial disputes

## Getting Started

This project is currently in initial setup phase.

## Architecture

- Frontend: Next.js with TypeScript
- Backend: FastAPI with Python
- Database: PostgreSQL with pgvector
- Caching: Redis
- AI Models: Local Ollama integration
