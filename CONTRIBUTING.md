# Contributing to Justice Companion

First off, thank you for considering contributing to Justice Companion! This project exists to help those who cannot afford legal representation, and every contribution helps make justice more accessible.

## Code of Conduct

By participating in this project, you agree to abide by our code of conduct: Be respectful, inclusive, and focus on helping those in need.

## How Can I Contribute?

### Reporting Bugs

Before creating bug reports, please check existing issues to avoid duplicates. When you create a bug report, include:

- A clear and descriptive title
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Screenshots if applicable
- Your environment (OS, Node version, etc.)

### Suggesting Enhancements

Enhancement suggestions are tracked as GitHub issues. When creating an enhancement suggestion, include:

- A clear and descriptive title
- A detailed description of the proposed enhancement
- Why this enhancement would be useful to users
- Possible implementation approach

### Pull Requests

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Make your changes
4. Run tests (`npm test`)
5. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
6. Push to the branch (`git push origin feature/AmazingFeature`)
7. Open a Pull Request

### Development Process

1. **Setup Development Environment**
   ```bash
   git clone https://github.com/yourusername/justice-companion.git
   cd justice-companion/justice-companion-app
   npm install
   npm start
   ```

2. **Code Style**
   - Use meaningful variable names
   - Comment complex logic
   - Keep functions small and focused
   - Follow existing patterns in the codebase

3. **Testing**
   - Write tests for new features
   - Ensure existing tests pass
   - Test on multiple platforms if possible

### Areas We Need Help

- 🌍 **Translations** - Help make the app accessible in more languages
- 📚 **Legal Templates** - Contribute templates for different jurisdictions
- 🎨 **UI/UX Improvements** - Make the interface more intuitive
- 📖 **Documentation** - Improve guides and tutorials
- 🐛 **Bug Fixes** - Help us squash bugs
- ♿ **Accessibility** - Ensure the app is usable by everyone

### Legal Contributions

If you're a legal professional:
- Review legal information accuracy
- Contribute jurisdiction-specific guidance
- Help create legal document templates
- Provide feedback on legal workflows

**Note:** All legal content must include appropriate disclaimers about not constituting legal advice.

## Recognition

Contributors will be recognized in our README and release notes. Your work helps provide access to justice for those who need it most.

## Questions?

Feel free to open an issue with your question or reach out through GitHub Discussions.

Thank you for helping make justice accessible to everyone! ⚖️
