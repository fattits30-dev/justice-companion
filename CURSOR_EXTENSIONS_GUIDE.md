# 🎯 Essential Extensions for Cursor/VS Code

## 🚀 How to Install Extensions in Cursor

1. **Open Extensions Panel**:
   - Press `Ctrl+Shift+X`
   - Or click the Extensions icon in sidebar

2. **Quick Install All Recommended**:
   - Go to Extensions tab
   - Click "Filter" icon → "Recommended"
   - Install all with one click

## ⭐ Must-Have Extensions (Install These First!)

### 1. **ESLint** (`dbaeumer.vscode-eslint`)
   - ✅ Already configured in settings.json
   - Catches errors as you type
   - Auto-fixes on save

### 2. **Prettier** (`esbenp.prettier-vscode`)
   - ✅ Already configured
   - Auto-formats your code
   - Works on save

### 3. **React Snippets** (`dsznajder.es7-react-js-snippets`)
   - Type `rafce` → Creates React component
   - Type `useState` → Hook snippet
   - Type `useEffect` → Effect snippet

### 4. **GitLens** (`eamodio.gitlens`)
   - See who wrote each line
   - View commit history inline
   - Git blame annotations

### 5. **Path IntelliSense** (`christian-kohler.path-intellisense`)
   - Auto-completes file paths
   - Essential for imports

## 🤖 AI-Powered Extensions (For Cursor)

### **GitHub Copilot** (`github.copilot`)
- Already built into Cursor!

## 🎨 Recommended Theme & Icons

## ⚡ Quick Install Commands

Run these in Cursor's terminal:

```bash
# Install all recommended at once
code --
code --install-extension esbenp.prettier-vscode
code --install-extension dsznajder.es7-react-js-snippets
code --install-extension eamodio.gitlens
code --install-extension christian-kohler.path-intellisense
code --install-extension pkief.material-icon-theme
```
